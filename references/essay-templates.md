# Essay Templates

## ARGUE - Argumentative Essay

### Template
```
[Hook: Startling statistic / Rhetorical question / Bold statement]

[Thesis statement: I strongly believe that...]

[Body 1 - First supporting point]
To begin with, [topic sentence]. For instance, [example]. This clearly demonstrates that [analysis]. Moreover, [additional evidence].

[Body 2 - Second supporting point]
Furthermore, [topic sentence]. Take [specific case] as an example. [Analysis of example]. Consequently, [implication].

[Body 3 - Counter-argument + rebuttal (optional but impressive)]
Admittedly, some people might argue that [opposing view]. However, [rebuttal with evidence]. Therefore, [concession does not weaken the main argument].

[Conclusion]
In conclusion, [restate thesis in different words]. It is high time that [call to action or final thought]. Only through [method] can we [desired outcome].
```

### Example Opening Lines
- "The question of whether [topic] has sparked heated debate among..."
- "It is widely acknowledged that [general statement], yet the issue of [specific topic] remains controversial."
- "Nowadays, an increasing number of people are concerned about [issue]."

## NARR - Narrative Essay

### Template
```
[Hook: Sensory detail or dialogue]

[Set the scene: When, where, who]

[Build-up: Events leading to the main point]

[Climax: The turning point or key moment]

[Resolution: What happened after]

[Reflection: What you learned or how it changed you]
```

### Useful Narrative Phrases
- "Little did I know that..."
- "Looking back, I realize that..."
- "That moment changed my perspective forever."
- "If I could turn back time, I would..."
- "It was then that I understood the true meaning of..."

## DESC - Descriptive Essay

### Template
```
[Opening: Overall impression using sensory language]

[Sight: What you see - colors, shapes, movements]

[Sound: What you hear - ambient sounds, voices]

[Smell/Taste: Olfactory and gustatory details]

[Touch: Textures, temperatures, physical sensations]

[Closing: How the place/thing makes you feel]
```

### Sensory Vocabulary
- **Sight**: gleaming, vibrant, misty, silhouette, panoramic
- **Sound**: murmuring, thunderous, deafening, whispering, resonating
- **Smell**: fragrant, pungent, musty, aromatic, refreshing
- **Touch**: velvety, rugged, slippery, coarse, silky

## EXPO - Expository Essay

### Template
```
[Introduction: Define the topic and state purpose]

[Body 1: Aspect 1 + explanation + example]

[Body 2: Aspect 2 + explanation + example]

[Body 3: Aspect 3 + explanation + example]

[Conclusion: Summary + significance]
```

### Expository Sentence Starters
- "The primary cause of [phenomenon] is..."
- "There are several factors that contribute to..."
- "One notable example of this is..."
- "This phenomenon can be attributed to..."

## LETTER - Letter Writing

### Formal Letter Template
```
[Sender's Address]
[Date]

[Recipient's Address]

Dear [Sir/Madam / Mr./Ms. Last Name],

[Opening paragraph: Purpose of writing]

[Body paragraphs: Details and requests]

[Closing paragraph: Expected action or response]

Yours sincerely/faithfully,
[Name]
```

### Informal Letter Template
```
[City, Date]

Dear [First Name],

[Opening: Personal greeting + reason for writing]

[Body: Share news, ask questions, respond to their news]

[Closing: Plans, wishes, promises to write again]

Best wishes / Take care / Love,
[Your name]
```
