# Vocabulary Upgrade Bank

## Replacing "Good/Bad" Words

| Common | Advanced Alternatives |
|--------|----------------------|
| think | contend, maintain, assert, reckon, am inclined to believe |
| important | crucial, vital, indispensable, paramount, of great significance |
| good | beneficial, favorable, advantageous, positive, constructive |
| bad | detrimental, adverse, harmful, negative, destructive |
| big | substantial, considerable, immense, enormous, vast |
| small | negligible, minimal, marginal, trivial, insignificant |
| many | a multitude of, numerous, a plethora of, countless |
| show | demonstrate, illustrate, indicate, reveal, highlight |
| help | facilitate, contribute to, promote, assist, bolster |
| get | obtain, acquire, attain, secure, procure |
| use | utilize, employ, apply, harness, leverage |
| make | produce, create, generate, construct, fabricate |
| say | state, declare, assert, emphasize, point out |
| need | require, necessitate, demand, call for |
| give | provide, supply, offer, yield, furnish |

## Degree Adverbs

| Common | Advanced |
|--------|----------|
| very good | exceptionally good, remarkably beneficial |
| very bad | exceedingly harmful, profoundly detrimental |
| more and more | an increasing number of, a growing proportion |
| sometimes | on occasion, from time to time, periodically |
| always | invariably, without exception, at all times |
| never | under no circumstances, on no account, at no point |

## Linking Phrases

### Cause
- owing to, on account of, due to, as a consequence of
- this stems from, this can be attributed to, this results from

### Effect
- consequently, as a result, thereby, hence, thus
- this leads to, this gives rise to, this culminates in

### Contrast
- nevertheless, on the other hand, conversely, whereas, albeit
- despite this, notwithstanding, in spite of, regardless

### Addition
- furthermore, moreover, in addition, what is more, besides
- equally important, of equal significance, additionally

### Example
- for instance, to illustrate, a case in point, specifically
- such as, namely, that is to say, to be more precise

### Conclusion
- in conclusion, to sum up, all things considered, in a nutshell
- taking everything into account, on the whole, overall

## Sentence Templates with Advanced Vocab

1. "It is universally acknowledged that [topic] plays a pivotal role in [context]."
2. "A multitude of factors contribute to [phenomenon], among which [factor] is particularly noteworthy."
3. "The ramifications of [issue] extend far beyond [immediate effect], encompassing [broader impact]."
4. "Not only does [A] [verb], but it also [additional verb], thereby [result]."
5. "The correlation between [A] and [B] has been extensively documented, suggesting that [conclusion]."
