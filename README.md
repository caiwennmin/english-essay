# English Essay Skill

A skill for generating high-scoring English essays for school homework assignments.

## What it does

Writes natural, human-looking English essays that avoid AI detection patterns. Covers the five common essay types:

- **Argumentative** — agree/disagree, opinion, should we...
- **Narrative** — a story, experience, once...
- **Descriptive** — describe, a place, my favorite...
- **Expository** — explain, how to, the importance of...
- **Letter** — formal/informal letter writing

## Features

- Structured templates for each essay type
- 200+ advanced vocabulary upgrades (replace basic words with sophisticated alternatives)
- 10 advanced grammar patterns (inverted sentences, subjunctive mood, participial phrases...)
- Anti-AI detection tips (sentence variation, contractions, personal anecdotes)
- Proper transition word banks to avoid repetition

## Usage

Ask me to write an English essay for school homework. I'll:

1. Identify the essay type from your prompt
2. Confirm topic, word count, and requirements
3. Apply the appropriate template and vocabulary
4. Output the essay with a structure summary

### Examples

```
Write an argumentative essay about whether social media is good for teenagers, 300 words
```

```
Write a descriptive essay about my favorite season
```

```
Write a formal letter to the school principal about improving the library
```
