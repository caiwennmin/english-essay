# English Essay Writing Skill (School Homework High-Score Edition)

## Core Principle

Write essays that look human, not AI-generated. Teachers detect robotic patterns. This skill produces natural, varied, high-scoring writing.

## Workflow

1. Identify essay type (argumentative / narrative / descriptive / expository / letter)
2. Confirm topic, word count, and any teacher requirements
3. Use appropriate template from `references/essay-templates.md`
4. Apply vocabulary from `references/vocab-upgrade.md`
5. Output the essay with a brief structure summary

## Essay Types & Keywords

| Type | Trigger Words | Template |
|------|--------------|----------|
| Argumentative | "agree/disagree", "opinion", "should we" | ARGUE |
| Narrative | "a story", "experience", "once" | NARR |
| Descriptive | "describe", "a place", "my favorite" | DESC |
| Expository | "explain", "how to", "the importance of" | EXPO |
| Letter | "write a letter", "formal/informal letter" | LETTER |

## High-Score Rules

### Structure (Non-Negotiable)
- **Introduction**: Hook + thesis statement (1 paragraph)
- **Body**: 2-3 paragraphs, each with topic sentence + evidence + analysis
- **Conclusion**: Restate thesis + final thought (1 paragraph)

### Vocabulary Upgrade
Replace common words with advanced synonyms. See `references/vocab-upgrade.md` for the full list.

Quick swaps:
- think → contend, maintain, argue
- important → crucial, vital, indispensable
- good → beneficial, favorable, advantageous
- bad → detrimental, adverse, harmful
- more and more → an increasing number of, a growing proportion
- because → owing to, on account of, due to
- very → exceedingly, remarkably, extraordinarily

### Grammar Patterns That Impress
1. **Inverted sentences**: "Not only does technology enhance efficiency, but it also fosters creativity."
2. **Subjunctive mood**: "If I were to choose, I would opt for..."
3. **Participial phrases**: "Having considered both sides, I firmly believe that..."
4. **Concessive clauses**: "Although some may argue otherwise, the evidence overwhelmingly suggests that..."
5. **Appositive phrases**: "Albert Einstein, a renowned physicist, once stated that..."
6. **Parallel structure**: "We should read more, think critically, and communicate effectively."

### Transition Words (Avoid Repetition)

**Adding**: furthermore, moreover, in addition, what is more, besides
**Contrasting**: nevertheless, on the contrary, conversely, whereas, albeit
**Concluding**: in conclusion, to sum up, all things considered, in a nutshell
**Exemplifying**: for instance, to illustrate, a case in point, specifically
**Cause/Effect**: consequently, as a result, thereby, hence, thus

## Anti-AI Detection Tips

- Vary sentence length (mix short punchy sentences with longer complex ones)
- Use occasional contractions in informal essays (don't, can't, it's)
- Include personal anecdotes or specific examples
- Avoid starting every paragraph with "In addition" or "Furthermore"
- Use discipline-specific vocabulary when appropriate
- Add rhetorical questions sparingly for engagement
